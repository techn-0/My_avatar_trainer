# 나의 아바타 트레이너

**나의 아바타 트레이너**는 사용자의 동작을 인식하여 운동을 측정하는 웹 서비스입니다.  
사용자의 운동 기록이 저장되어 아바타가 기록을 재연하게 되고,  
사용자는 자신의 기록을 따라하는 아바타와 경쟁하여 더 많은 횟수를 기록하면 승리합니다.
